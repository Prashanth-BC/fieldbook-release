# Fieldbook Release

This repository contains the release artifacts for the Fieldbook Sync system.

## Installation

### Client (Obsidian Plugin)
1. Go to `common/client`.
2. Run: `./install.sh /path/to/your/obsidian/vault`

### Server (Sync Hub)
1. Copy the `common/server` folder to your Raspberry Pi.
2. Run: `sudo ./install.sh`
